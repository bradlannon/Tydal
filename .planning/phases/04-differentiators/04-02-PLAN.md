Create input/gyroscope.js:
- Import filterFX from engine/effects.js and getActiveSynth from engine/instruments.js
- Module state: _active = false, _hasPermission = false, smoothedBeta = 0, smoothedGamma = 0
- JITTER SMOOTHING (UX-03): Use exponential moving average (EMA) with alpha = 0.15. On each deviceorientation event: smoothedBeta = alpha * raw.beta + (1 - alpha) * smoothedBeta. Same for gamma. This eliminates jitter while keeping response under 100ms.
- PARAMETER MAPPING (UX-01):
  - gamma (left/right tilt, range -45 to +45): Map to filter cutoff 200Hz–8000Hz. Use exponential mapping: freq = 200 * Math.pow(40, (gamma + 45) / 90). Call filterFX.frequency.rampTo(freq, 0.05) for smooth audio parameter changes.
  - beta (forward/backward tilt, range -30 to +30): Map to pitch bend in cents: cents = (beta / 30) * 200. So full forward = +200 cents (2 semitones up), full backward = -200 cents. Call getActiveSynth().set({ detune: cents }).
  - Clamp gamma to [-45, 45] and beta to [-30, 30] before mapping.
- iOS PERMISSION (UX-02): Export requestPermission() async function:
  - Check if DeviceOrientationEvent.requestPermission exists (iOS 13+)
  - If yes, call it and check result === 'granted'
  - If no (Android/desktop), permission is implicit — return true
  - Set _hasPermission on success
- Export initGyroscope() that adds the deviceorientation event listener (only if _hasPermission and _active)
- Export setGyroActive(bool) to enable/disable without removing listener (when disabled, reset detune to 0 and filter to 4000Hz)
- Export isGyroActive() returning _active

Create ui/gyro-panel.js:
- Export initGyroPanel(containerEl) that creates a small control section
- Contains:
  - A toggle button "Gyro: Off" / "Gyro: On" (styled like existing panel-header buttons)
  - On first enable: call requestPermission(). If iOS and denied, show brief inline message "Motion access denied — check Settings > Safari"
  - On enable: call initGyroscope() then setGyroActive(true)
  - On disable: call setGyroActive(false)
- If DeviceOrientationEvent is undefined (desktop without gyro), show "Gyro: N/A" in disabled state

Update index.html:
