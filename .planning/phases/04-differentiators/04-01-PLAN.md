  <action>
Create engine/visualizer.js:
- Import Tone and masterVolume from effects.js
- Create a Tone.js Analyser (or raw AnalyserNode via Tone.getContext().rawContext.createAnalyser()) with fftSize=2048
- Connect masterVolume to the analyser node (tap off the signal, do NOT insert it into the chain — use masterVolume.connect(analyserNode) as a fan-out)
- Export initAnalyser() that creates and connects the node (call after audio started)
- Export getFrequencyData() returning Uint8Array of frequency bins
- Export getTimeDomainData() returning Uint8Array of time-domain samples
- Export detectTransient() that compares current RMS energy against a rolling average (window of ~10 frames). Return true when current energy exceeds average by >1.8x (drum hit threshold). Store previous energy values in a module-level ring buffer.

Create ui/visualizer-ui.js:
- Export initVisualizer(containerEl) that creates a canvas element inside containerEl
- Canvas should be full-width of container, height 120px, with CSS class "visualizer-canvas"
- Create a mode toggle button (Spectrum / Waveform) placed above the canvas
- requestAnimationFrame render loop:
  - SPECTRUM MODE (VIZ-01): Draw frequency bars. Map each FFT bin to a vertical bar. Color bars using HSL where hue = (bin index / total bins) * 300 to create a rainbow gradient from bass (red) to treble (purple).
  - WAVEFORM MODE (VIZ-02): Draw oscilloscope line. Use time-domain data to draw a continuous path across the canvas. COLOR the waveform based on detected pitch (VIZ-03): estimate fundamental frequency from zero-crossing rate of time-domain data. Map frequency to HSL hue: C2(65Hz)=0(red) through C6(1046Hz)=300(purple). Default to teal (#00FFCC as HSL) when no note is playing (low amplitude).
- TRANSIENT GLOW (VIZ-04): When detectTransient() returns true, trigger a glow effect — set canvas globalAlpha to flash white overlay that decays over 150ms (track transientGlow float 0->1, decay by 0.05/frame). Draw a semi-transparent white rect over the entire canvas at that alpha before drawing the normal visualization.
- Store mode ('spectrum'|'waveform') in module variable, toggle button switches it

Update index.html:
- Add <div id="visualizer"></div> between the instrument container and the synth-panel div

Update styles.css:
- Add .visualizer-canvas { width: 100%; height: 120px; border-radius: 8px; background: rgba(0,0,0,0.3); }
- Add #visualizer { margin: 12px 0; position: relative; }
- Style the mode toggle button to match existing panel-header style (dark background, teal accent text, small font)

Update app.js:
- Import { initVisualizer } from './ui/visualizer-ui.js'
- Call initVisualizer(document.getElementById('visualizer')) after initSequencerUI
