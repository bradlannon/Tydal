<task type="auto">
  <name>Task 1: Patch serialization engine with localStorage and URL sharing</name>
  <files>engine/preset-storage.js</files>
  <action>
Create engine/preset-storage.js:

Define a patch schema (plain object, JSON-serializable):
```
{
  version: 1,
  name: string,
  timestamp: number,
  synth: {
    factoryPreset: string | null,   // If from a factory preset, store name
    type: 'subtractive' | 'fm',
    params: { ... }                  // Full synth params (oscillator, envelope, filter, etc.)
  },
  effects: {
    reverb: { decay, wet },
    delay: { delayTime, feedback, wet },
    distortion: { distortion, wet },
    filter: { frequency, type, Q },
    channel: { volume, pan },
    vibrato: { frequency, depth, wet },
    tremolo: { frequency, depth, wet },
    filterLFO: { frequency, min, max, type, enabled: bool }
  }
}
```

Export captureCurrentPatch(name):
- Read synth state from getActiveSynth().get() — extract oscillator.type, envelope (ADSR), filter, filterEnvelope, detune
- Detect if FM: check if getActiveSynth().get() has harmonicity property
- Read effects state from each exported effect node: reverb.get(), delay.get(), distortion.get(), filterFX.get(), channel.get(), vibrato.get(), tremolo.get(), filterLFO (read frequency/min/max/type and check if state is started)
- For filterLFO.enabled: check if filterLFO.state === 'started'
- Return patch object

Export savePatch(patch):
- Read existing patches from localStorage key 'tydal-patches' (JSON array, default [])
- Append patch with timestamp
- Write back to localStorage
- Cap at 50 saved patches (remove oldest if over)

Export loadPatch(patch):
- If patch.synth.factoryPreset and it exists in PRESETS, use applyPreset(name)
- Else: create new Tone.PolySynth with appropriate voice type (Tone.Synth or Tone.FMSynth) using patch.synth.params, then switchInstrument(newSynth)
- Apply effects: for each effect key in patch.effects, call the corresponding node's .set() method (reverb.set({decay, wet}), etc.)
- For filterLFO: set params then start/stop based on enabled flag via setLFO('filterLFO', { ...params, enabled })

Export listPatches():
- Return parsed array from localStorage 'tydal-patches', default []

Export deletePatch(index):
- Remove patch at index from the array, save back

Export patchToURL(patch):
- JSON.stringify the patch
- btoa() to base64 encode (use encodeURIComponent for URL safety on non-ASCII)
- Return current origin + pathname + '#patch=' + encoded string

Export patchFromURL():
- Read location.hash
- If it contains '#patch=', extract the base64 string after '='
- atob() decode, JSON.parse
- Validate version field exists
- Return patch object or null
