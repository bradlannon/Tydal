What was built:
- Subtractive synth engine with waveform selection, ADSR, filter, voice stealing
- Full effects chain (reverb, delay, distortion, filter, LFO modulation)
- 7 factory presets (3 subtractive, 3 FM, 1 pluck) with preset selector

- Velocity sensitivity on touch pads

- Scale lock mode (6 scales, all 12 keys)

How to verify — start local server: `python3 -m http.server 8080`, open http://localhost:8080 in Chrome:

Success Criterion 1 — Polyphony and voice stealing:
1. Hold 8 keyboard keys simultaneously (Z X C V A S D F) — all 8 notes should sound
2. While holding those 8 keys, press Q — oldest note should release and Q should sound
3. Release all keys — no stuck notes

Success Criterion 2 — Velocity sensitivity on touch:
1. Open DevTools mobile mode, tap a pad slowly — quieter note
2. Tap a pad with fast swipe — louder note
3. Volume difference should be clearly audible

Success Criterion 3 — MIDI keyboard (if available):
1. Connect USB MIDI keyboard, play a note — correct pitch
2. Play softly and loudly — velocity maps to volume
3. If no MIDI: console shows "not supported" info, no error

Success Criterion 4 — Chromatic layout and octave shift:
1. Pads show C3, C#3, D3, D#3, ... D#4 (16 chromatic semitones)
2. Click octave up — shifts to C4 range
3. Click octave down — returns to C3 range

Success Criterion 5 — Effects and controls:
1. Select "FM Piano" preset — sound changes dramatically
2. Adjust ADSR attack to max — slow fade-in
3. Increase reverb wet to 0.8 — heavy reverb
4. Increase delay wet to 0.5 — echo repeats
5. Adjust filter cutoff — tone brightens/darkens

Bonus — Scale lock:
1. Set scale to C Pentatonic — pads show only C D E G A
2. Set scale to Off — full chromatic returns
