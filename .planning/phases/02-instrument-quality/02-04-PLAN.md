  - Preset selector: &lt;select&gt; dropdown with all preset names. On change, call applyPreset(value).
  - Waveform selector: &lt;select&gt; with sine, square, sawtooth, triangle. On change, call setSynthParam({ oscillator: { type: value } }). Disable when FM preset is active (FM oscillator is always sine).
  - ADSR sliders: 4 range inputs for attack (0.001-2, step 0.01), decay (0.01-2, step 0.01), sustain (0-1, step 0.01), release (0.01-3, step 0.01). On input, call setSynthParam({ envelope: { [param]: value } })
  - Filter controls: cutoff slider (50-10000, step 10, log-scale feel), resonance slider (0.1-20, step 0.1). On input, call setSynthParam({ filter: { frequency: value } }) and setSynthParam({ filter: { Q: value } })
  - Filter type selector: lowpass, highpass, bandpass. On change, call setSynthParam({ filter: { type: value } })

Create ui/fx-panel.js:
- Import { reverb, delay, distortion, filterFX, channel, vibrato, tremolo, filterLFO, setLFO } from '../engine/effects.js'
- Export function initFXPanel(containerEl): creates and appends a collapsible panel with:
  - Reverb: wet slider (0-1), decay slider (0.1-10). On input, reverb.wet.value = val / reverb.decay = val
  - Delay: wet slider (0-1), time selector (1/4, 1/8, 1/16 as '4n','8n','16n'), feedback slider (0-0.9). On input, delay.wet.value / delay.delayTime.value / delay.feedback.value
  - Distortion: amount slider (0-1). On input, distortion.distortion = val, distortion.wet.value = val > 0 ? 1 : 0
  - Filter FX: cutoff slider (50-10000), resonance slider (0.1-20). On input, filterFX.frequency.value / filterFX.Q.value
  - Channel: volume slider (-40 to 0 dB), pan slider (-1 to 1). On input, channel.volume.value / channel.pan.value
  - Vibrato: depth slider (0-1), rate slider (1-10 Hz). On input, vibrato.depth.value / vibrato.frequency.value / vibrato.wet.value = depth > 0 ? 1 : 0
  - Tremolo: depth slider (0-1), rate slider (1-10 Hz). On input, tremolo.depth.value / tremolo.frequency.value / tremolo.wet.value = depth > 0 ? 1 : 0

Both panels should be collapsible via a header toggle (click header to expand/collapse body). Default state: collapsed.
