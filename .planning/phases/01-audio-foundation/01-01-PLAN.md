<task type="auto">
  <name>Task 2: Build audio engine, signal chain, overlay module, and app bootstrap</name>
  <files>engine/audio-engine.js, engine/instruments.js, engine/effects.js, ui/overlay.js, app.js</files>
  <action>
Create the audio engine modules and bootstrap wiring:

1. **engine/audio-engine.js** (AUDIO-01):
   - `let _started = false;`
   - `export async function ensureAudioStarted()` — if `_started` and `Tone.context.state === 'running'`, return immediately. Otherwise call `await Tone.start()`, set `_started = true`. Wrap in try/catch, console.error on failure.
   - iOS interrupted state recovery: `Tone.getContext().rawContext.addEventListener('statechange', ...)` — if state is no longer `'running'`, set `_started = false` and dispatch a custom event `'audio-interrupted'` on `document` (overlay.js will listen for this).
   - Import `* as Tone from 'tone'`.
   - CRITICAL: This is the ONLY module that calls `Tone.start()`. No other module creates an AudioContext.

2. **engine/instruments.js** (AUDIO-04, AUDIO-06):
   - Import `* as Tone from 'tone'`.
   - Create and export `warmPad` as `new Tone.PolySynth(Tone.Synth, { maxPolyphony: 8, options: { oscillator: { type: 'sawtooth' }, detune: -8, envelope: { attack: 0.02, decay: 0.1, sustain: 0.85, release: 0.4 }, filter: { type: 'lowpass', frequency: 2800, rolloff: -12 }, filterEnvelope: { attack: 0.05, decay: 0.2, sustain: 0.5, release: 0.8, baseFrequency: 200, octaves: 3.5 } } })`.
   - The release: 0.4 value provides the anti-click fade (AUDIO-06).
   - Export a `noteOn(note)` function: calls `warmPad.triggerAttack(note, Tone.now())`. Use `Tone.now()` — never setTimeout (AUDIO-05).
   - Export a `noteOff(note)` function: calls `warmPad.triggerRelease(note, Tone.now())`. Use `Tone.now()` (AUDIO-05).

3. **engine/effects.js** (AUDIO-02):
   - Import `* as Tone from 'tone'` and `{ warmPad }` from `'./instruments.js'`.
   - Create `export const masterVolume = new Tone.Volume(-6).toDestination();` — -6dB default headroom.
   - Wire: `warmPad.connect(masterVolume);`
   - This establishes the signal chain: PolySynth -> Volume -> Destination (AUDIO-02).

4. **ui/overlay.js**:
   - Import `{ ensureAudioStarted }` from `'../engine/audio-engine.js'`.
   - Get reference to `#audio-overlay` DOM element.
   - Export `showAudioOverlay()` — sets `overlay.hidden = false`.
   - Export `hideAudioOverlay()` — sets `overlay.hidden = true`.
   - On overlay click: call `await ensureAudioStarted()`, then `hideAudioOverlay()`. Use `{ once: false }` because interrupted state can bring it back.
   - Listen for custom `'audio-interrupted'` event on `document`: call `showAudioOverlay()`.
   - On init: if `Tone.context.state !== 'running'`, show the overlay. Otherwise hide it.

5. **app.js** (bootstrap):
   - Import `'./engine/effects.js'` (side effect: wires signal chain).
   - Import `{ masterVolume }` from `'./engine/effects.js'`.
   - Import `'./ui/overlay.js'` (side effect: sets up overlay listeners).
   - Wire volume slider: `document.getElementById('volume-slider').addEventListener('input', (e) => { masterVolume.volume.rampTo(Number(e.target.value), 0.05); });`
   - Console.log `'Tydal ready'` to confirm load.
   - NOTE: Input handlers (keyboard.js, touch.js) and pad grid rendering are NOT imported here yet — Plan 02 adds those.

IMPORTANT: All timing calls use `Tone.now()` — zero instances of `setTimeout` in any engine/ or input/ file (AUDIO-05). ES modules via importmap (INTG-04).
