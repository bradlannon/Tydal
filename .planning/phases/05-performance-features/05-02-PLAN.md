<task type="auto">
  <name>Task 1: Note repeat engine</name>
  <files>engine/note-repeat.js</files>
  <action>
Create engine/note-repeat.js:

Module state:
- let enabled = false
- let rate = '8n' (default: eighth notes)
- const activeRepeats = new Map() — note → intervalId

Rate-to-ms conversion helper:
- rateToMs(rate): calculate from getBPM()
  - '4n' = (60000 / bpm) * 1
  - '8n' = (60000 / bpm) * 0.5
  - '16n' = (60000 / bpm) * 0.25
  - '32n' = (60000 / bpm) * 0.125

Export startRepeat(note, velocity = 0.8):
- If not enabled or activeRepeats.has(note), return
- Calculate interval from rateToMs(rate)
- Set interval: each tick calls noteOff(note), then after 5ms setTimeout calls noteOn(note, velocity)
- Store intervalId in activeRepeats map

Export stopRepeat(note):
- Get intervalId from activeRepeats, clearInterval it
- Remove from map
- Call noteOff(note)

Export setRepeatRate(newRate):
- Update rate
- For each active repeat: clear old interval, start new one with recalculated interval
- Store velocity per repeat for restart (expand map value to { intervalId, velocity })

Export setRepeatEnabled(val): set enabled flag. If disabling, stop all active repeats.
Export isRepeatEnabled(): return enabled
Export getRepeatRate(): return rate
  </action>
