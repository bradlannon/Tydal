  <action>
Create engine/randomizer.js:

Import: setSynthParam from instruments.js, reverb/delay/distortion/filterFX/vibrato/tremolo from effects.js, captureCurrentPatch/loadPatch from preset-storage.js

Module state:
- const variations = [null, null, null, null] — 4 slots for patch snapshots

Helper: randomChoice(arr) — return random element from array
Helper: randomRange(min, max) — return random float in range
Helper: expRandom(min, max) — exponential distribution random (bias toward lower values)

Export randomizePatch():
- Synth params (applied via setSynthParam):
  - oscillator.type: randomChoice(['sine', 'square', 'sawtooth', 'triangle'])
  - envelope.attack: randomRange(0.001, 0.5)
  - envelope.decay: randomRange(0.05, 0.8)
  - envelope.sustain: randomRange(0.1, 1.0)
  - envelope.release: randomRange(0.05, 2.0)
  - filter.frequency: expRandom(200, 8000) — exponential distribution for musical spread
  - filter.Q: randomRange(0.5, 6)
  - detune: randomRange(-20, 20)

- Effects (60% chance each stays off, when on use conservative ranges):
  - Track totalWet = 0, cap at 1.5 ("chaos budget")
  - Reverb: 60% chance wet=0 (off), else wet=randomRange(0.1, 0.6), decay=randomRange(1, 5)
  - Delay: 60% chance wet=0, else wet=randomRange(0.1, 0.4), feedback=randomRange(0.1, 0.5)
  - Distortion: 60% chance wet=0, else distortion=randomRange(0.05, 0.4), wet=randomRange(0.2, 0.8)
  - For each effect, add its wet to totalWet. If totalWet exceeds 1.5, scale down remaining effects' wet values or skip
  - Apply via each effect node's .set() method (e.g., reverb.set({ wet, decay }))
  - Vibrato: 40% chance on — wet=randomRange(0.2, 0.7), depth=randomRange(0.05, 0.3), frequency=randomRange(2, 8)
  - Tremolo: 30% chance on — wet=randomRange(0.2, 0.6), depth=randomRange(0.1, 0.5), frequency=randomRange(2, 10)

Export saveVariation(slot):
- slot is 0-3
- variations[slot] = captureCurrentPatch(`Variation ${slot + 1}`)

Export loadVariation(slot):
- If variations[slot] is null, return false
- loadPatch(variations[slot])
- return true

Export listVariations():
- Return array: variations.map(v => v ? v.name : null)
