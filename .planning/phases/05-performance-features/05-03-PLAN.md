  <action>
Create engine/macros.js:

Import effects from engine/effects.js: reverb, delay, distortion, filterFX, vibrato, tremolo

Define MACROS object with 4 macro definitions. Each macro has a name and array of param mappings:
```
{
  Darkness: [
    { target: filterFX, param: 'frequency', min: 4000, max: 200 },
    { target: reverb, param: 'wet', min: 0, max: 0.8 },
    { target: delay, param: 'wet', min: 0, max: 0.5 },
  ],
  Grit: [
    { target: distortion, param: 'distortion', min: 0, max: 0.8 },
    { target: distortion, param: 'wet', min: 0, max: 1 },
    { target: filterFX, param: 'Q', min: 1, max: 8 },
  ],
  Motion: [
    { target: vibrato, param: 'wet', min: 0, max: 1 },
    { target: vibrato, param: 'depth', min: 0, max: 0.5 },
    { target: tremolo, param: 'wet', min: 0, max: 1 },
    { target: tremolo, param: 'depth', min: 0, max: 0.6 },
  ],
  Space: [
    { target: reverb, param: 'wet', min: 0, max: 0.9 },
    { target: reverb, param: 'decay', min: 1, max: 8 },
    { target: delay, param: 'wet', min: 0, max: 0.6 },
    { target: delay, param: 'feedback', min: 0.2, max: 0.7 },
  ],
}
```

Track current values: macroValues = { Darkness: 0, Grit: 0, Motion: 0, Space: 0 }

Export applyMacro(name, value):
- value is 0-1 (from slider)
- For each mapping in MACROS[name]:
  - Calculate interpolated value: min + (max - min) * value
  - Apply to target: check if target[param] is a Tone Signal (has .value property)
    - If Signal: target[param].rampTo(interpolated, 0.05)
    - If direct property (like distortion.distortion): target[param] = interpolated
    - Special case: filterFX.frequency is a Signal, use .rampTo
    - Special case: filterFX.Q — use filterFX.Q.value = interpolated (it's a Signal)
    - For reverb.decay: direct assignment (not a Signal)
  - Actually, simplest approach: use target.set({ [param]: interpolated }) which handles both Signal and plain props via Tone's built-in .set() method
- Store value in macroValues

Export getMacroValue(name): return macroValues[name]
Export MACROS (for UI to read macro names)
