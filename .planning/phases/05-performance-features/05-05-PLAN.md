  <name>Task 1: Preset browser UI with preview flow</name>
  <files>ui/preset-browser.js</files>
  <action>
Create ui/preset-browser.js:

Import: applyPreset, getPresetNames from presets.js
Import: captureCurrentPatch, loadPatch, listPatches from preset-storage.js
Import: noteOn, noteOff, releaseAll from instruments.js
Import: ensureAudioStarted from audio-engine.js

Module state:
- let backupPatch = null — saved before first preview
- let currentPreview = null — name of currently previewing preset
- let previewTimeouts = [] — for cleanup

Export initPresetBrowser(sheetContentEl):
  Build UI inside sheetContentEl:

  1. "Factory Presets" heading (h3 or panel-heading class)
  2. For each preset in getPresetNames():
    - Create .preset-item div with name label
    - On click: handlePreview('factory', presetName)

  3. "My Patches" heading
  4. For each patch in listPatches():
    - Create .preset-item div with patch.name label
    - On click: handlePreview('user', index)
  5. If no user patches: show "No saved patches" placeholder text

  6. Action buttons at bottom:
    - "Load" button (class panel-btn): confirm current preview, close sheet
    - "Cancel" button (class panel-btn): restore backup, close sheet

  Internal functions:

  handlePreview(type, identifier):
    - If no backup exists: backupPatch = captureCurrentPatch('_browser_backup')
    - releaseAll() to stop any playing notes
    - If type === 'factory': applyPreset(identifier)
    - If type === 'user': loadPatch(listPatches()[identifier])
    - Update UI: remove .previewing from all items, add to clicked item
    - currentPreview = identifier
    - Play test chord: await ensureAudioStarted(), then:
      - noteOn('C4', 0.6), noteOn('E4', 0.6), noteOn('G4', 0.6)
      - After 800ms timeout: noteOff('C4'), noteOff('E4'), noteOff('G4')
      - Store timeout for cleanup

  confirmSelection():
    - backupPatch = null (discard backup)
    - currentPreview = null
    - Clean up preview timeouts
    - Close the preset-browser-sheet (dispatch event or call global closeSheet)

  cancelSelection():
    - If backupPatch: loadPatch(backupPatch)
    - backupPatch = null
    - currentPreview = null
    - releaseAll()
    - Clean up preview timeouts
    - Close sheet

  Also export a refresh function so the browser can be rebuilt when patches change.
  </action>
