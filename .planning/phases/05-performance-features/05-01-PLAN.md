<task type="auto">
  <name>Task 1: Pad expression engine module</name>
  <files>engine/pad-expression.js</files>
  <action>
Create engine/pad-expression.js:

- Track active expressions per pointerId: `Map<pointerId, { note, padRect, smoothX, smoothY }>`
- EMA_ALPHA = 0.3 constant for smoothing
- Export startExpression(pointerId, note, padRect):
  - Store entry in map with smoothX=0.5, smoothY=0.5 (center position)
- Export updateExpression(pointerId, clientX, clientY):
  - Get entry from map, return if not found
  - Normalize clientX/clientY to 0-1 within padRect (clamped)
  - Apply EMA smoothing: smoothX = alpha * rawX + (1-alpha) * smoothX (same for Y)
  - X → detune: (smoothX - 0.5) * 400 → setSynthParam({ detune })
  - Y → filter: 200 + (1 - smoothY) * 5800 → filterFX.frequency.rampTo(freq, 0.05)
  - (Y inverted: top of pad = high frequency, bottom = low)
- Export stopExpression(pointerId):
  - Remove from map
  - Reset: setSynthParam({ detune: -8 }) (match default)
  - Reset: filterFX.frequency.rampTo(4000, 0.1)
- Export hasExpression(pointerId):
  - Return boolean for whether pointer has active expression
  </action>
